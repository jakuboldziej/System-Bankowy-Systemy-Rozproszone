# System Bankowy - Instrukcja Uruchomienia

Projekt składa się z aplikacji serwerowej oraz klienckiej (GUI oparty na CustomTkinter).

## Wymagania

- Python 3.10 lub nowszy

## Instalacja i uruchomienie (Windows)

1. Stwórz wirtualne środowisko i je aktywuj:
   ```bash
   python -m venv venv
   .\venv\Scripts\activate
   ```
2. Zainstaluj wymagane biblioteki:
   ```
   pip install -r requirements.txt
   ```
3. Uruchom serwer (w pierwszym oknie terminala):
   ```
   python server.py
   ```
4. Uruchom klienta (w drugim oknie terminala):
   ```
   python client.py
   ```
